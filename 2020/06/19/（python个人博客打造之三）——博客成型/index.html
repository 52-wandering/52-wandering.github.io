bootstrap是一个比较优秀的前端UI，由html，css，js等写成，可以制作网页。

bootstrap的使用

推荐一个bootstrap的中文文档网址：http://code.z01.com/v4/docs/

简单地使用可以通过CDN来使用。即复制相应的链接到html文件中即可。

在使用过程中，可以直接复制bootstrap文档中的案例代码，再自己组合修改即可，通常的做法是复制元素内的class内容，如：

bootstrap上手很简单，难在一些网页的自适应布局和组件的布置，可以细心琢磨一下，工具使用：浏览器的下按F12键。效果：

  1. 点击第一行左侧的箭头，移动鼠标到网页，可以选取到网页的元素
  2. 点击箭头右侧的两个方形，可以切换屏幕的大小和设备属性，即可以通过点击此，来切换电脑屏幕和小设备（平板/手机）屏幕显示，切换到小屏幕后，可以设置小屏幕的尺寸，来达到查看自适应的效果。如下所示：
